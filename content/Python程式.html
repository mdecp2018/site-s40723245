Python程式

def ：定義

len ：(全名lenght) 列出有幾個字串

input ：經由處理將數字變成字串

# ：用來註解，在其可打文字說明且不影響程式進行

"""Steins Gate"""  ：前後加3個雙引號或單引號可將2個單字換行

Strins /n Gate ：同樣有換行的作用

Python專業術語說明

data type 資料型態(分類)

浮點數(小數)

字串：任意的文字內容，前後需加雙引號或單引號

布林值：表達正確(Ture)或錯誤(False)，字首大寫

可變列表（List）：有順序、可變動的資料集合

固定列表(Tuple)：有順序、不可變動的資料集合

字典：鍵值對(key-Value Pair)的集合

集合：無順序的資料集合

變數(Bariable)：可用來存放資料的自訂名稱

↓↓~以下是Python程式的範例~↓↓

# 資料: 程式的基本單位
# 數字
3456
3.5
# 字串
"啊哈哈哈"
"Steins Gate"
# 布林值
True
False
# 有順序、可動的列表 List
[3,4,5]
["steins","Gate"]
# 有順序、不可動的列表 Tuple
(3,4,5)
("Steins","Gate")
# 集合
{3,4,5}
{"Steins","Gate"}
# 字典 Dictionary
{"Steins Gate":"命運石之門","data":"資料"}
# 變數: 用來儲存資料的自訂名稱
# 變數名稱=資料
x=3
# print(資料)
print(x)
x=True # 取代舊的資料
print(x)
x="Steins Gate"
print(x)
x={3,4,6} # 集合 Set
print(x)
# 數字運算
x=3*6
print(x)
x+=1 #x=x+1 #將變數中的數字 +1
print(x)
# 字串運算
# 字串會對內部的字元編號(索引), 從0開始算起
s="Steins"
print(s[1:4])
# 有序可變動列表 List
grades=[37,56,61,80,87]
grades[1:4]=[] # 連續刪除列表中從編號 1 到 編號 4(不包括)的資料
grades=grades+[13,15]
length=len(grades) # 取得列表的長度len(列表資料)
print(length)
grades[0]=45 #把 45 放到列表中的第一個位置
print(grades)
# 有序不可變動列表 Tuple
