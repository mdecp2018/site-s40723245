git基本程式

git clone  將遠端檔案下載至近端

git commit -m "initial add"  提交近端檔案並命名為initial add

git config --global user.email "電子信箱"   第一次上傳檔案需要登入，儲存至home

git config --global user.name "s+學號"  同上(但非必須)

git push 將近端檔案推上遠端

dir 顯示目錄的資料夾

python wsgi.py 動態擷取系統(打開近端網站必須)

git checkout gh-pages  導出gh-page

git pull 近端遠端同步

git status 查詢有更動的資料

mkdir test  在目錄裡建立資料夾並命名為test

cls 清理終端機的畫面

git branch dev 新增branch叫做dev

git log 顯示所有的版本

git log --oneline --graph --all 用一行顯示有更新過的全部圖檔紀錄
