程式語言

程式語言是人類和機器溝通的橋樑，透過程式語言，人們可以更精確地對電腦下達指令，達到目標。

git基本程式

git clone  將遠端檔案下載至近端

git commit -m "initial add"  提交近端檔案並命名為initial add

git config --global user.email "電子信箱"   第一次上傳檔案需要登入，儲存至home

git config --global user.name "s+學號"  同上(但非必須)

git push 將近端檔案推上遠端

dir 顯示目錄的資料夾

python wsgi.py 動態擷取系統(打開近端網站必須)

git checkout gh-pages  導出gh-page

git pull 近端遠端同步

git status 查詢有更動的資料

mkdir test  在目錄裡建立資料夾並命名為test

cls 清理終端機的畫面

git branch dev 新增branch叫做dev

git log 顯示所有的版本

git log --oneline --graph --all 用一行顯示有更新過的全部圖檔紀錄

Python程式

def ：定義

len ：(全名lenght) 列出有幾個字串

input ：經由處理將數字變成字串

# ：用來註解，在其可打文字說明且不影響程式進行

"""Steins Gate"""  ：前後加3個雙引號或單引號可將2個單字換行

Strins /n Gate ：同樣有換行的作用

Python專業術語說明

data type 資料型態(分類)

浮點數(小數)

字串：任意的文字內容，前後需加雙引號或單引號

布林值：表達正確(Ture)或錯誤(False)，字首大寫

可變列表（List）：有順序、可變動的資料集合

固定列表(Tuple)：有順序、不可變動的資料集合

字典：鍵值對(key-Value Pair)的集合

集合：無順序的資料集合

變數(Bariable)：可用來存放資料的自訂名稱

week1

開啟近端網站教學

第一週主要是教我們python的基本操作，用可攜式的p37程式進行近端上傳到遠端的更新作業，點開p37進入start.bat，然後在指令頁面先打隨身碟或電腦裡的槽，例:在c槽打"c:"、在d槽打"d:"，(以下用c槽舉例)，打完c:後下一行會出現隨身碟裡面的p37位置，這時候要先回去c槽最前面的目錄，打cd \就能回到最前面的目錄。在隨身碟新增資料夾，資料夾名稱可以自取，假設將資料夾命名為tmp，接下來的步驟要把自己的倉儲下載到近端，在指令頁面打git clone 網址，這樣tmp資料夾裡面就會有一份自己的倉儲檔案，再來就可以開啟自己的近端網站，點進倉儲檔案裡，然後在指令頁面打cd tmp/倉儲檔案名稱，下一行打python wsgi.py，這樣就能從瀏覽器進去個人的近端網站。
